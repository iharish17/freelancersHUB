Freelancer Marketplace

A modern freelancer marketplace web application inspired by Fiverr, built using React (Create React App).
The platform enables clients to discover freelancers and services, while freelancers manage offerings and requests through a role-based dashboard system.
This project focuses on clean architecture, scalable frontend structure, and clear user workflows, making it suitable for real-world deployment, hackathons, or portfolio presentation.
